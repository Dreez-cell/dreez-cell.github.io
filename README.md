## dreez-cell.github.io

World-class news, hip hop, and sports aggregator. Powered by 80+ curated RSS feeds, lightning-fast loading, mobile-first accessibility, progressive web app features, advanced SEO, and revenue-optimized ads.

## Features

- **80+ RSS feeds**: Most up-to-date news, hip hop, and sports from the best sources globally.
- **Accessibility**: Screen reader friendly, keyboard navigation, ARIA labels, color contrast, skip links, and more.
- **Performance**: Loads fast, lazy-loads images, fetches feeds in parallel, and is bandwidth-efficient.
- **SEO**: Structured data, sitemap, canonical URLs, meta tags, Open Graph/Twitter Cards, and indexable article links.
- **Progressive Web App**: Works offline, installable, manifests, and icons.
- **Ads**: Optimized placements for improved revenue while maintaining a quality user experience.
- **Disqus Comments**: World-class embedded integration.
- **Article Search**: Find articles from all feeds instantly.
- **Sitemap**: Auto-generated for all loaded articles.
- **Indexing**: All feeds and articles are easily indexable for search engines.
- **Mobile-First**: Fully responsive, touch-optimized. dreez-cell.github.io
